# Journal Style 规则

## Abstract（摘要）
- **严禁数学符号**
- 150-200 词
- 结构：动机 → 方法 → 主要发现 → 政策含义
- 不引用文献

## Introduction（引言）
- **严禁数学符号**
- 用事实/现象开头
- 明确 "本文做了什么"
- 预览主要结论（用文字）
- Related Literature 段落
- 最后一段说明论文结构

## Proposition 格式
- 每个 Proposition 后必须有 2-3 段经济学解释
- 第 1 段：核心直觉（为什么？）
- 第 2 段：机制分析（通过什么渠道？）
- 第 3 段：文献联系或政策含义

## 写作用语
- 用 "We show that..." 不用 "We can see that..."
- 用 "It follows that..." 不用 "It is obvious that..."
- 避免 "Interestingly,..." — 直接说结论
- 模型描述用现在时，文献回顾用过去时

## 格式
- 12pt, double-spacing, 1-inch margins
- 图表编号连续，caption 详细
- 交叉引用用 `\ref{}` 和 `\cite{}`
- 图形使用 PDF 矢量格式
