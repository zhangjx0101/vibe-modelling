# Quality Gates 规则

## 扣分标准（100 分制，从 100 开始扣）

| 类别 | 扣分项 | 分值 |
|---|---|---|
| Blocking | 公式 LaTeX 编译失败 | -100 |
| Blocking | FOC/SOC 验证失败 | -100 |
| 数学 | Wolfram 未验证的符号声明 | -50 |
| 数学 | 缺少数值验证（无 /verify） | -20 |
| 数学 | SOC 未验证 | -20 |
| 文献 | 参考文献不存在（编造） | -30 |
| 事实 | 特征事实无数据源 | -15 |
| 写作 | Proposition 无经济学解释 | -15 |
| 写作 | Abstract/Intro 含数学符号 | -10 |
| 格式 | 交叉引用错误 | -5/处 |
| 语言 | 语法错误 | -2/处 |

## 门控阈值

| 分数 | 门控 | 行动 |
|---|---|---|
| < 70 | 阻止 | 必须修复 Blocking issues |
| 70-79 | 警告 | 允许但列出改进建议 |
| 80-89 | 可提交 | 达到投稿标准 |
| 90-94 | 良好 | 高质量 |
| ≥ 95 | 优秀 | 目标水平 |

## 探索性工作

测试替代模型设定等探索性工作，门槛降至 60/100：
- 不需要完整 Proposition 格式
- 不需要语言润色
- FOC/SOC 验证仍然必须（数学正确性不妥协）

## 评分时机

- 每个 Phase 完成后可选评分
- 完整论文提交前必须评分
- 分数记录到 `quality_reports/reviews/` 中
