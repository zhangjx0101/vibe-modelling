# 文献检索

系统性地检索和整理与研究问题相关的文献。

## 工作流程

### Step 1：关键词检索
- 用 WebSearch 搜索核心关键词
- 搜索 Google Scholar、SSRN、NBER
- 覆盖：直接相关、方法论相关、背景文献

### Step 2：分类整理

```markdown
## 直接相关（模型/问题最接近）
1. [Author (Year)] - [标题]
   - 模型: [简要设定]
   - 主要结论: [1-2 句]
   - 与本文关系: [我们的差异/改进]

## 方法论相关（使用类似工具）
...

## 背景文献（提供动机/事实）
...
```

### Step 3：草拟 Related Literature

按主题组织（不是按时间罗列）：
- "关于 X 的研究..."
- "关于 Y 的研究..."
- "本文与上述文献的区别在于..."

### Step 4：验证
- 所有引用交给 `lit-verifier` Agent 验证真实性
- **绝不编造引用** — 不确定的标记 [需验证]

## 注意事项
- 不要只列论文，要说明每篇与本文的关系
- 重点突出本文填补的 gap
- 引用格式遵循目标期刊
