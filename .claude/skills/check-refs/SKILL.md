# 参考文献验证

验证论文中所有引用的真实性，防止 AI 编造引用。

## 验证流程

### 逐条检查

对每条引用验证：

| 验证项 | 方法 |
|---|---|
| 作者名 | WebSearch 确认 |
| 年份 | 与实际发表年份对比 |
| 期刊/出版物 | 确认期刊存在且论文发表在该期刊 |
| 卷号/页码 | 交叉验证 |
| DOI | 如有，确认可访问 |
| 引用内容 | 确认我们引用的结论与原文一致 |

### 输出

```markdown
# Literature Verification Report

## 验证结果
| # | 引用 | 作者 ✓ | 年份 ✓ | 期刊 ✓ | 内容 ✓ | 状态 |
|---|---|---|---|---|---|---|
| 1 | Author (Year) | ✓ | ✓ | ✓ | ✓ | VERIFIED |
| 2 | Author (Year) | ✓ | ✗ | ✓ | ? | NEEDS CHECK |

## 问题引用
### [引用 X]
- 问题: [具体描述]
- 建议: [如何修正]
```

## 重要原则

- **宁可标记 [需验证] 也不要放过可疑引用**
- 如果无法通过 WebSearch 验证，标记为 UNVERIFIED
- 特别注意工作论文（working paper）的状态 — 可能已正式发表
