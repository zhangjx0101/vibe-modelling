# 事实核查

验证论文中引用的事实、数据、趋势的准确性。

## 检查范围

### 需要验证的声明类型
- 行业趋势："AI 成本在过去十年大幅下降"
- 统计数字："全球 AI 市场规模达到 X 亿美元"
- 行业事实："从制造业到金融服务都在采用 AI"
- 政策事实："某国政府补贴了 X 行业"
- 历史事实："混合寡头在欧洲公共事业中普遍存在"

### 不需要验证的
- 数学定义和推导结果
- 文献中的已知理论结论（由 check-refs 负责）
- 模型假设（由 econ-reviewer 负责）

## 验证流程

1. **提取**：从 .tex 中提取所有事实性声明
2. **搜索**：用 WebSearch 查找数据源
3. **验证**：对比论文声明与实际数据
4. **补充**：建议更精确的数据或更权威的数据源

## 输出

```markdown
# Fact Check Report

| # | 声明 | 数据源 | 状态 |
|---|---|---|---|
| 1 | "AI 成本下降了 90%" | Stanford AI Index 2025 | VERIFIED |
| 2 | "市场规模 500 亿" | 未找到来源 | UNVERIFIED |

## 建议修改
### 声明 2
- 原文: "..."
- 问题: 数字可能不准确
- 建议: 引用 [具体报告] 的数据 "..."
```
